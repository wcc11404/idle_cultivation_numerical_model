# 游戏背景设定集（修仙世界观）

## 1. 世界概述
此界名为“玄灵界”，天地由“三境一隙”构成：
- 凡尘境：万民生息、王朝更替、百业并立。
- 仙门境：宗门、世家与散修盟共同维持修行秩序。
- 边荒境：妖兽繁衍、灵脉紊乱、秘境频现，是历练与机缘并存之地。
- 归墟隙：天地裂隙与古战场遗迹，传闻连通上古遗域。

灵气由地脉升腾、天象牵引、万灵反哺而循环。修士以纳气、炼体、悟道三法并进，凡俗也能通过灵植、矿脉、符器产业参与修行文明。修士并非脱离尘世，而是以“护境、守脉、镇灾”为天职之一。

### 凡尘境与中州的关系
凡尘境在本文中是一个“文明层级概念”，不是某一座城、某一片地图或单一行政区。它指的是以凡俗秩序为主体、并与修行体系长期共存的人间世界整体。

凡尘境不是修行世界的附庸，而是玄灵界的根基层。它由王朝、州府、郡县、乡镇与无数村落构成，承担粮药生产、矿材开采、手工业制造与人口延续。没有凡尘境的稳定供给，修行体系便无法长期维持。

中州则是凡尘境与仙门境最深度重叠的中枢区域：一方面，中州拥有最成熟的凡俗治理与道路水网；另一方面，中州汇聚宗门议席、世家祖地与商会总号，是修行秩序的调度中心。可以把两者理解为“地基与枢纽”的关系：凡尘境提供底盘，中州完成汇流与放大。

在玄灵界的日常运转里，凡尘境与中州不是上下割裂，而是层层嵌套。凡人通过灵田、药圃、驿路与工坊参与灵材供给；修士通过护城、治灾、镇妖与守脉反哺人间秩序。二者互为因果，共同构成“可持续的修行文明”。

## 2. 历史脉络（简版）
### 上古纪
上古诸族并立，妖庭与人族并争，天地灵机旺盛但秩序混乱。最终爆发“归墟大战”，大量古宗湮灭，诸多传承断代。

### 宗门纪
幸存传承重建为宗门体系，立下“护脉盟约”：谁占灵脉，谁护疆域；谁得传承，谁守规矩。历练制度、试炼塔制度与破境礼逐步成形。

### 当世纪
宗门、世家、散修盟三方制衡，凡俗王朝在修行秩序庇护下维持稳定。边荒异动频繁，古遗迹复苏，历练从“个人修行”演化为“全境防务的基础机制”。

## 3. 修行体系世界观
玄灵界现行修行主干为九大境界，每一境界分十层（十层为大圆满）。这种“九境十层”的制度，不仅用于衡量战力，也用于划分职责、资源配额与护境权限。

### 3.1 九大境界与十层结构
- 炼气期（一层至十层）：引气入体、稳固经络，是入道与识法阶段。修士从“能修”走向“可战”。
- 筑基期（一层至十层）：重塑根基、定脉定魂，开始具备长期驻守与独立执行任务能力。
- 金丹期（一层至十层）：丹成内转，法力循环稳定，能在中型战场或复杂任务中承担主责。
- 元婴期（一层至十层）：神魂外显、感知大增，开始具备跨域机动与高难度镇压能力。
- 化神期（一层至十层）：神识与法则感知显著提升，可参与区域级阵务与大规模调度。
- 炼虚期（一层至十层）：由实返虚，擅长处理裂隙、残阵、异相等高风险环境问题。
- 合体期（一层至十层）：身魂法意高度合一，常担任宗门中坚、边关统领与跨势力调停者。
- 大乘期（一层至十层）：道行近满，负责顶层防务、灾厄封镇与传承校正。
- 渡劫期（一层至十层）：承天劫而证大道，十层圆满者具备“飞升资格”，可尝试叩开上界天门。

在文化语义上，每个境界的十层并非简单数字递增，而是“初立—渐稳—精深—大圆满”的完整修行周期。修士从一层开始修术，至十层修心，最终完成本境界的责任交接。

### 3.2 飞升仙界的铺垫
玄灵界并不把渡劫期十层视为终点，而视为“人间修行的毕业线”。

现世共识认为，飞升并非单次战斗胜负，而是三重条件共振：
- 道基完整：九境修行无重大断层，神魂与法体保持统一。
- 因果清明：未触禁术重罪，护境功绩与道心证言可追溯。
- 天门契机：天地灵机与个人命数在特定时机共鸣，方可引动“上界接引”。

因此，后续“仙界篇”可以自然承接为：渡劫圆满修士在玄灵界完成职责清算后，进入更高法则密度的上界体系，展开新的道统、势力与资源竞争。

### 3.3 大体系并行：大道百艺的世界定位
除境界主线外，玄灵界修行文明建立在多条“百艺副轴”上，且这些体系将直接决定后续玩法扩展空间：
- 炼丹：连接灵草、矿材与境界突破，是“资源转化效率”的核心。高阶丹道可影响宗门补给与战后恢复能力。
- 符箓：以一次性法意封装见长，决定战前准备与临场应变上限。常与法宝、阵法形成联动。
- 阵法：承担护山、守城、封印、聚灵等基础设施职能，是区域治理与大规模作战的底层技术。
- 法宝：作为个人战力与宗门底蕴的综合载体，连接炼器、符箓、冶炼等多系统，代表修士长期积累。

这意味着玄灵界的成长不是“只冲境界”，而是“境界主线 + 百艺体系”共同驱动。后续无论扩展洞府、宗门还是上界内容，都可沿此框架自然展开。

## 4. 地理区域总览（中州地图设想）
若自九天俯瞰中州，其形如一枚被群山与大泽环抱的灵印：中央是政令与法度并行的天衡城群，东侧江海相接、舟商不绝，西北高岭连绵、古宗林立，西南湿热迷雾、异族杂居，北境风雪切骨、铁骑与剑修共守边关，南部丘陵与平原交错，是凡俗王朝与修真商路最密集的地带。

中州并非“单城统治”，而是“多中心共治”。宗门掌修行秩序，世家掌地脉与乡里，商会掌流通命脉，散修盟掌灰区缓冲，几方在地图上互相咬合，形成了既紧张又稳定的平衡。

### 4.1 中州核心地带：天衡城群与三川灵带
中州核心位于三条主灵河交汇处，城池呈环状分布，内城多为议事、藏典、铸器与大阵中枢，外城则是坊市、渡口、驿站与客卿府。这里地势平缓，却是灵压最稳之地，适合建立大型阵法和长期传承。

核心地带由“天衡盟议”维持基本秩序，正道宗门在此设有常驻议席，世家派驻家老，商会设总号，散修盟设调停堂。凡俗百姓与低阶修士混居，街巷里既有卖符纸的老铺，也有专司灵植保育的官坊，日常气质是“庄重而繁忙”。

### 4.2 东境水网：沧澜江海与百港商路
东境由大江入海形成巨型水网，支流如织、湖泊密布，水雾常年不散。这里盛产水属性灵材与航运人才，大小港埠像星点一样沿岸排开，夜间灯火与阵纹浮标连成一线。

主导势力为“沧浪商盟”与数个沿海世家。前者控制跨域运输、海贸契券与保险规约，后者掌握河口灵闸与潮汐阵。东境的修士风格讲究“快、稳、算”，重视机动与补给，不恋死战。常见风险并非正面冲突，而是水妖潮、沉舟怨气与走私引发的黑市争斗。

### 4.3 西北山脊：太玄群岭与古宗山门
西北是中州海拔最高的区域，群峰直入云海，峡谷深切，古木与寒松并生。多条古灵脉在山腹交织，适合闭关、铸剑、养兽与守阵。高地地势天然易守难攻，是传统宗门最偏爱的立派地形。

此地由数个老牌宗门共同维持边岭秩序，彼此虽有道统之争，却在外敌来犯时协同极快。西北修士普遍重“戒律与师承”，行事风格克制硬朗。自然环境上的最大威胁是雪崩灵潮、崖隙妖巢与古战残阵复苏，常常一夜改写山道格局。

### 4.4 西南雾泽：离瘴泽国与异民聚落
西南地势低陷，沼泽、雨林与火山余脉交错，昼夜温差小、湿热漫长，瘴雾层叠不散。这里并非荒蛮之地，而是中州最复杂的“多族共居带”：药民、猎民、蛊师支系和外来散修长期杂处。

主要话语权在地方盟约而非单一宗门。散修盟西南分堂在此极强，负责跨族调停与猎场界线划分。西南的生态危险高度环境化，毒藤、雾兽、寄生灵虫与地热裂隙共同构成风险。能在此立足的修士，往往更擅长识地、识毒、识风向，而非单纯比拼爆发。

### 4.5 北境寒原：玄霜长关与边军剑府
北境是风雪主宰之地，冰原、冻河、黑松林和断崖防线层层推进。这里与外域接壤，长期承受妖潮与流寇压力，因此形成“军镇 + 剑府”并行结构：军镇守民生，剑府守战线。

统领力量以边军世家与战修宗门为主，辅以中州盟议的轮值援军。北境社会气质朴直坚硬，重承诺、重协作、重战后抚恤。常见生灵有霜背狼群、雪原角鹿、寒羽凶隼，也有受寒煞侵蚀后的异变兽群。这里的每一条补给线都决定着一场守城战能否撑到天明。

### 4.6 云稷平川：云稷沃野与人间烟火（玩家初始出生地）
玩家初始出生地设定在中州南麓“云稷平川”的稷安里一带。此地离天衡城群约四百里，走官道需两至三日，既不脱离中州主秩序，也不会被高烈度战事直接波及，是“可见修行、可近人间”的起点区域。

术语约定：本文中的“南麓”是“中州南麓”的简称，指代南部多区域带状地理总称；“云稷平川”是南麓范围内面积最大、人口最密集的核心区域之一。

云稷平川的核心地理坐标可概括为“一城、两河、三镇、四野”：
- 一城：云稷城（州府级城镇），南麓税仓、药市与票号总柜所在。
- 两河：澄水河与南曲河，在城东二十里汇流，灌溉主粮带。
- 三镇：稷安里（新手聚落）、百草镇（药材集散）、渡平码头（舟运与护运队驻点）。
- 四野：青梧林地、白石丘原、百草山、乌藤旧矿带。

此外，云稷平川南偏西约二十二里有一处小型石灰岩洞，地方俗称“破境草洞穴”。洞腔纵深不过数十丈，分三岔窄道，潮湿却通风，常年有微弱地脉回暖，偶可孕生破境草。因洞体狭小、可采点位有限，破境草产量始终不高，通常每旬仅得寥寥数株，需由百草会馆与州府药监共同限采，避免涸泽而渔。

若以稷安里为中心，周边地形可直接对应到可命名的地域：
- 西行三十里：青梧林地。林冠高而疏，地面苔厚，晨雾常在卯时前后最重。常见云角鹿、灰纹狐、溪鳞蛇，偶有“木灵迷途”传闻：旅人夜行会被重复林径困住，需循风铃木声才可出林。
- 西北四十五里：白石丘原。丘陵与砾石地交错，草短风急，适合放牧与短途巡猎。常见短鬃山彘、岩雀群、泥背灵龟。民间流传“白石夜火”：雨后子夜常见无主磷光沿古驿道漂移，被认为是古驿卒残念守路。
- 南行五十里：百草山。山脉自云稷平川南缘起势，向西南绵延数百里，山体温润、雾线低垂，溪谷与缓坡间遍布野生药圃。此地少有凶恶怪物盘踞，常见的是性情温顺的青羽鹭、花背灵兔、短尾岩鹿。由于灵草种类繁杂、外形近似，山中长期存在大量未经辨认的灵草，百草镇药农与学徒常成队入山采药辨草。百草镇正是依山而兴，地方民众将百草山视作赖以生存的神山。更深处的阴坡与云雾谷地间，亦流传偶见高品质稀有灵草的消息，被视作云稷一带最稳妥也最讲眼力的采药地。
- 东南六十里：乌藤旧矿带。废弃矿井与黑藤共生，地表塌陷点多。常见疫鼠群、枯藤妖芽和受秽气侵染的小型矿灵。州府常派巡修队封井，坊间则传“井下旧炉”仍藏未熄地火，可炼异材。

此地灵息特征偏“温和、绵长、可驯化”。白昼清风暖阳，夜间地脉微鸣，利于初修调息与凡俗耕作并行。正因灵压稳定，南麓被称为“养人之地”：不以惊险著名，却最适合让人把根基打实。

南麓地方秩序由“世家 + 商会 + 州府”三方共治，且分工明确：
- 陆氏（云稷陆家）：祖地在云稷城北十二里陆家圩，专长灵田轮养与水渠阵。
- 沈氏（南泽沈家）：主宅在渡平码头上游八里，掌河运、渡契与护运舟队。
- 许氏（青岭许家）：居于百草山东北麓，经营药圃、矿材分拣与草木鉴定。

活跃商会同样构成玩家早期社会生态：
- 万和行：云稷城南市主号，覆盖粮药、符纸、基础法器，是最常见的民生商号。
- 四海票号：城东银栈总柜，负责跨州结算与押运担保，连接中州核心金融网络。
- 百草会馆：百草镇会所，主导药材分级、丹材竞拍与医馆供给，左右本地药价。

因此，云稷平川不是抽象的新手区，而是一块有坐标、有路网、有传闻、有产业分工的真实土地。玩家从这里出发，先学会“此界如何生活”，再逐步触及“此界如何修仙”。

### 4.7 势力分布总述（按地图关系）
- 正道宗门：集中在西北山脊与中州核心议席，掌修行法统与大型防御体系。
- 修真世家：沿东境水网、云稷平川与北境关隘分布，掌地方治理与长期资源经营。
- 散修盟：在西南雾泽与核心城群外环最活跃，承担灰区调停与机动任务承接。
- 市井商会：以东境港路和核心城群为双枢纽，向南北两端铺开供给网络。
- 魔道残脉：多潜伏于边缘裂隙、废弃矿脉与失管黑市，不占明面疆域，却会沿商路与灾带渗透。

从空间逻辑上看，中州像一张被灵脉与商路共同织成的网：中心定法度，四境供资源，边关承压力，灰区做缓冲。任何一处区域动荡，最终都会沿着这张网传导到整个修行秩序。

### 4.8 南麓试练塔（原无尽塔）
云稷平川诸镇将南麓试练塔视作“南线公试场”，既用于检验地方修士战备，也用于筛选护运队与巡守队骨干。

南麓试练塔位于云稷平川与白石丘原交界的古驿高地，被视为上古试炼制度留存至今的“可重复验证场”。它并不对应单一地貌，而是由层层阵台、回廊和镜面灵障拼合而成的半人工战场。

与野外历练不同，南麓试练塔强调“持续稳定性”而非单次爆发。入塔修士会在不同层面对速度、承伤、输出节奏与临场判断进行连续检验。塔中敌手采用中州现行生态模板轮替，现主要投影为青梧与白石两系生物（青纹狐、青鳞蛇、云角鹿、白纹林狼、白蛇、短鬃山彘）

因此，南麓试练塔在玄灵界中的定位更接近“战斗品控与道心压测”设施：它既是各势力衡量后进修士稳定性的公共标尺，也是修士自证成长路径是否扎实的试金石。

## 5. 主要势力设定
### 正道宗门
以护境与传承为核心，重视门规、师承与历练履历。掌握主要试炼制度与大型护山阵法。

### 修真世家
依托血脉与祖地经营，擅长长期资源管理与人脉联结。常与宗门合作镇守重要地脉或商路。

### 散修盟
由无门无派修士结成，提供任务撮合、纠纷仲裁与临时互助。对边荒情报最敏锐，是历练生态的重要补充。

### 市井商会
负责灵材流通、丹器交易与跨域运输，维系凡俗与修真之间的供给链。商会中立但受盟约法度约束。

### 魔道残脉
并非单一组织，而是若干偏离盟约的隐秘流派。其核心特征是“以捷径换代价”，常触及禁术红线，被各方联防清剿。

### 势力关系（文字简图）
正道宗门居中立规，世家与商会维持资源流，散修盟连接边荒执行层；魔道残脉长期渗透边缘地带，形成“秩序—供给—执行—破坏”四股张力。

## 6. 习俗与禁忌
### 收徒礼
讲究“问心、问责、问缘”。师徒关系绑定护道责任，不得只授术不授德。

### 破境礼
修士破境需向师门或盟约备案，宣誓承担对应职责。此礼既是庆典也是契约。

### 历练契约
进入历练区前，默认签署生死自负与战利分配条款；组队需明示主责与撤离判定。

### 交易规约
丹药、灵材、法器交易需遵守来源可追溯原则。黑货、禁术材料与污染灵物严禁入市。

### 禁术红线
以寿元掠夺、神魂污染、凡俗献祭、地脉强抽为四大禁术红线。触犯者视同“破盟”，可被跨势力联合追缉。

## 7. 历练文化定位
历练在玄灵界有三重意义：
- 修行意义：检验道心、补足实战、验证功法。
- 社会意义：以战养防，让低阶修士承担可控护境任务。
- 资源意义：通过规则化战斗与掉落体系，稳定灵材供给。

按功能可将历练区域分为：
- 成长区：低风险、可持续，用于境界初期磨炼。
- 资源区：产出明确、次数受限，用于破境关键材料获取。
- 试炼区：规则严苛、层级递进，用于综合能力验证与荣誉排序。

## 8. 命名原则附录（供后续区域改名使用）
为保证区域名称统一且可扩展，建议采用“地貌词 + 修行词 + 风险词”三段式语义框架：
- 地貌词：体现空间质感，如林、谷、原、泽、洞、塔、渊。
- 修行词：体现阶段或用途，如炼气、筑基、破境、试炼、问道。
- 风险词：体现危险层级，如外缘、内域、绝地、禁区、镇守。

命名约束建议：
- 同一成长线使用一致的层级词（如外缘/内域），避免混用。
- 资源区优先突出用途语义（如破境、灵草、矿脉），便于玩家理解。
- 试炼区优先突出规则语义（如层、关、阶、塔），强化挑战预期。

以上原则仅用于语义统一，不直接替代现有区域 ID 与数值配置。
